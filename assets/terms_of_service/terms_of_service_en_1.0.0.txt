By installing, using or registering in the Application, you (hereinafter referred to as the "User") agree to the terms of this Agreement and Privacy Policy. If you disagree, do not use the Application.

## 2. Age Restrictions

The Application is intended for persons aged 13 and older.

## 3. Registration and Authentication

Users can log into the Application in the following ways:

- Through Google account
- Through Apple ID
- By email (with account creation)

The Application gains access to:

- Username
- Email address
- IP address, device type and OS version (for diagnostics and abuse prevention)

This data is used only for authorization, personalization and communication with the User.

## 4. Processed Data

Users can voluntarily specify and save in the Application:

- Name, email, profile photo
- Geo-tags, fishing spots, routes
- Angler level, fishing types
- Fish quantity and weight
- Photos and notes
- Information on marker bottom maps

Data is stored in Firebase (Google Cloud) and is not transferred to third parties, except in cases provided by law.

## 5. Data Deletion

Users can delete:

- Individual notes, photos and maps
- Their account and all associated data (in personal cabinet)

After deletion, data recovery is impossible.

## 7. Used Permissions

For proper operation, the Application may request access to:

- GPS (for geo-tags and forecasts)
- Camera (for photos)
- Device storage (for saving photos and exporting data)
- Push notifications (for fishing reminders)

## 8. Offline Operation

In offline mode, users can:

- View previously downloaded notes
- Create new notes
- View and edit marker bottom maps

When internet connection is restored, new data is synchronized with cloud storage.
