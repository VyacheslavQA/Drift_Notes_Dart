РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Добро пожаловать в Drift Notes!

НАЧАЛО РАБОТЫ

Drift Notes - это ваш персональный журнал рыбалки. Приложение поможет вам:
- Вести записи о рыбалках
- Сохранять фотографии уловов
- Отслеживать статистику
- Планировать будущие поездки

СОЗДАНИЕ ЗАМЕТОК

1. На главном экране нажмите центральную кнопку с логотипом рыбки
2. Выберите тип рыбалки
3. Заполните информацию:
   - Место рыбалки
   - Дату и время
   - Используемые снасти
   - Добавьте фотографии
4. Сохраните заметку

ИСПОЛЬЗОВАНИЕ КАРТЫ

- Просматривайте все ваши точки рыбалки на карте
- Выбирайте новые места для будущих поездок
- Сохраняйте координаты интересных мест

ФУНКЦИИ ПОГОДЫ

- Отслеживайте текущую погоду
- Просматривайте прогноз погоды
- Данные о погоде автоматически сохраняются в заметках

ИСПОЛЬЗОВАНИЕ ТАЙМЕРОВ

- Настройте напоминания о смене приманки
- Установите таймеры для подкормки
- Создавайте любые другие напоминания

СИНХРОНИЗАЦИЯ

Все ваши данные автоматически синхронизируются между устройствами при наличии интернет-соединения.

ПОДДЕРЖКА
