Политика конфиденциальности приложения Drift Notes

Приложение Drift Notes (далее — «Приложение») уважает вашу конфиденциальность. В настоящей Политике конфиденциальности разъясняется, какие данные собираются, как они используются, где хранятся и какие права вы имеете как пользователь.

1. Какие данные мы собираем
Мы собираем следующие данные, которые вы добровольно предоставляете при использовании Приложения:
• Имя, адрес электронной почты, номер телефона (для регистрации и входа)
• Данные, полученные при входе через Google или по номеру телефона
• Геолокация (для отметки рыболовных мест на карте)
• Фото и текстовые заметки, связанные с рыбалкой
• Типы снастей, погодные условия, карты дна и прочие данные, связанные с рыбалкой
• Маркерные карты дна водоемов (глубины, рельеф, тип дна, названия водоемов)
• Настройки аккаунта (язык, единицы измерения и т. д.)
• IP-адрес и данные устройства (модель, ОС), исключительно для технической диагностики и предотвращения злоупотреблений
• Токены для push-уведомлений (для отправки напоминаний о рыбалке)
• Данные об использовании приложения (для аналитики и улучшения сервиса)
• Данные о сбоях приложения (для устранения ошибок)

2. Как мы используем эти данные
Собранные данные используются исключительно для:
• Работы и персонализации Приложения (юридическое основание: исполнение договора)
• Отображения информации на карте и в календаре (юридическое основание: исполнение договора)
• Анализа активности клева с помощью моделей OpenAI и Anthropic Claude (анализ выполняется без доступа к персональным данным) (юридическое основание: законный интерес)
• Хранения ваших заметок и данных в вашем аккаунте (юридическое основание: исполнение договора)
• Отправки push-уведомлений о напоминаниях и важных событиях (юридическое основание: согласие)
• Улучшения качества сервиса, устранения сбоев, обеспечения безопасности и предотвращения мошеннических действий (юридическое основание: законный интерес)
• Аналитики использования приложения для улучшения функциональности (юридическое основание: законный интерес)

3. Где хранятся данные
Все пользовательские данные хранятся в Firebase (Google Cloud), серверы которого расположены в США и других странах. При передаче данных в США применяются соответствующие меры защиты согласно решениям Европейской комиссии о достаточности.

Данные также временно хранятся локально на вашем устройстве для обеспечения работы в офлайн режиме. Эти локальные данные автоматически синхронизируются с облачным хранилищем при подключении к интернету.

Данные не передаются третьим лицам. Мы не продаем и не передаем ваши данные внешним организациям, за исключением случаев, предусмотренных законодательством (например, по требованию суда).

4. Обмен данными между пользователями
Вы можете добровольно делиться маркерными картами дна водоемов с другими пользователями приложения. При этом:
• Передаются только технические данные о рельефе дна (глубины, тип дна, названия водоемов)
• Персональная информация (имя, email, контакты) НЕ передается
• Обмен происходит только с вашего явного согласия
• Вы контролируете какие карты отправлять
• Переданные карты можно удалить из вашего аккаунта в любой момент
Юридическое основание: добровольное согласие пользователя.

5. Удаление данных
Вы можете удалить любую из своих заметок или записей в Приложении в любой момент. Также вы можете удалить свой аккаунт через интерфейс Drift Notes. После удаления аккаунта все ваши данные будут безвозвратно удалены из всех систем (Firebase Firestore, Firebase Storage, локальные данные) и не подлежат восстановлению.

6. Доступ для детей
Приложение может использоваться пользователями любого возраста, включая лиц младше 18 лет. Для пользователей младше 13 лет требуется согласие родителей или законных представителей. Мы не собираем специально информацию о возрасте, но обязуемся обеспечивать конфиденциальность и безопасность всех пользователей.

7. Безопасность данных
Передача и хранение данных осуществляется с использованием современных методов безопасности, включая HTTPS, аутентификацию, авторизацию и шифрование. Мы принимаем разумные меры для защиты ваших данных от утечки, утраты и несанкционированного доступа.

8. Разрешения устройства
Приложение запрашивает следующие разрешения:
• Местоположение — для отметки рыболовных мест на карте
• Камера — для создания фотографий рыбалки
• Хранилище — для сохранения фотографий и экспорта данных
• Уведомления — для напоминаний о рыбалке и важных событий
• Интернет — для синхронизации данных и загрузки карт
Вы можете отозвать эти разрешения в настройках устройства, что может ограничить функциональность приложения.

9. Используемые сторонние сервисы
Приложение использует следующие сторонние сервисы:
• Firebase (Google) — для хранения данных, аутентификации, аналитики и push-уведомлений
• Google Maps API — для отображения карты
• WeatherAPI — для загрузки данных о погоде
• OpenAI API и Anthropic Claude — для анализа клева (без доступа к персональным данным)
• Google Play Services, App Store Services, RuStore Services — для обработки оплаты, аналитики и доставки обновлений
• Firebase Analytics — для анализа использования приложения
• Firebase Crashlytics — для сбора данных о сбоях приложения

10. Права пользователя
В соответствии с GDPR вы имеете право:
• Получать информацию о хранимых данных (право на доступ)
• Исправлять неточные или неполные данные (право на исправление)
• Удалять отдельные данные или весь аккаунт (право на забвение)
• Экспортировать свои данные в структурированном формате (право на портируемость данных)
• Ограничивать обработку ваших данных в определенных случаях
• Возражать против обработки данных на основании законного интереса
• Отозвать согласие на обработку данных (что приведёт к удалению аккаунта)
• Прекратить использование Приложения в любой момент
• Подать жалобу в надзорный орган по защите данных

Для реализации этих прав обратитесь к нам по адресу support@driftnotesapp.com.

Данные аналитики и сбоев хранятся в обезличенном виде до 24 месяцев для улучшения качества сервиса.
